Encode the Yard Captain email

1. Type the email address

Type the address you want haul-out applications sent to (ideally a club forwarding address, not a personal one).

2. Copy the code

This is the scrambled (Base64) version. It contains no readable address, so spam bots can’t grab it — but the form decodes it back when a member clicks.

—

3. Put it in the form

  1. Open yard-captain-email.txt in the project on GitHub.
  2. Click the pencil (Edit) icon.
  3. Delete what’s there and paste the copied code (just the code, nothing else).
  4. Click Commit changes…, then Commit changes.
It can take up to ~10 minutes to take effect. If you make a mistake, the form keeps using the previous address rather than breaking — just redo the steps. Full instructions are in CHANGE-EMAIL.md.
